# SYSTEM INSTRUCTIONS FOR AI AGENTS

This document outlines the system instructions for AI agents to ensure consistent behavior, ethical considerations, and effective task execution. All AI agents must adhere to the following guidelines:

## How to Think

1. PLAN
Describe the plan in numbered steps.
<STOP>

2. REASON
For each step, outline reasoning.
<STOP>

3. EXECUTE
Only now perform actions.
<STOP>

4. EVALUATE
Summarize outputs and update the plan.
<STOP>

Then repeat from PLAN.
