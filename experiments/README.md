# Experiments  

This directory contains prototypes, autonomous agents, and one-off studies. Each substantial experiment should include its own README with a clear goal, approach, and how to run it.  

## Index  

### Sub-directories  
- **agents-system-prompt** – experiments with agent system-prompt engineering.  
- **browser project** – work on browser automation or agent-driven browsing.  
- **cnn** – convolutional-neural-network related experiments.  
- **electra** – explorations of the ELECTRA language model architecture.  
- **erebus** – sub-project with licensing guidance and agent documents.  
- **gpt-oss** – open-source GPT-style model experiments.  
- **i-robot** – robotics or embodied-AI explorations.  
- **mixis** – mixture-of-experts or multi-agent studies.  
- **muse** – creative AI projects inspired by muses.  
- **philosobot** – philosophical dialogue bots and reasoning agents.  
- **prometheus** – ambitious or large-scale experimental frameworks.  

### Individual scripts and data  
- **aeon.py** – prototype exploring long-term or cyclical processes.  
- **busy_beaver.py** – script dealing with Busy Beaver computational challenges.  
- **calliope.py** – creative-generation tool inspired by the muse Calliope.  
- **file analyzer.py** – utility for analyzing files or datasets.  
- **hyperion.py** – prototype relating to large-scale systems.  
- **interlinked.txt** – notes on interlinked concepts or systems.  
- **louvre.txt** – conceptual notes with an artistic metaphor.  
- **mirror test.py** – tests self-recognition or agent introspection.  
- **oss.py** – helper script for working with open-source software.  
- **panacea.jsonl** – JSON Lines dataset used for experiments.  
- **system optimizer.py** – tool for optimizing system configurations.  
- **tartarus.py** – deep experiment named after the mythological abyss.  
- **the ghost.json** – structured data for an experiment codenamed "The Ghost".  
- **trent.tex** – LaTeX document (notes or a paper draft).  
- **turing.txt** – notes relating to Turing machines or tests.  
- **turing_demo.py** – demonstration script associated with Turing experiments.
